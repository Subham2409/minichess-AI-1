# minichess-AI-1

## main build

### make 'build' directory

At the first time build, you should run

```
mkdir build
```

### build

First, move corrent directly to 'build'

```
cd build
```

update Makefile

```
cmake ..
```

build

```
cmake --build .
```

### execute

In 'build' directory

```
minichessAI
```

## test build

### build

First, move corrent directory to test

```
cd test
```

After that, do same process of main build

### execute

In 'test/build' directory

```
minichessAI_test
```
